## Step-by-step walkthrough

1. Create a project
Open Unity hub and press new project ![new project page](\assets\images\lessons\2023-01-25\DELETEME1.png)

Select 3d core for your project type. ![3d core](\assets\images\lessons\2023-01-25\DELETEME2.png)

2. Download necessary assets
The assets can be found at this [link](https://www.google.com/url?q=https://connect-prd-cdn.unity.com/20210507/bfd26de3-a68a-4a16-8cf6-8eacf2bb7f75/Prototype%25202%2520-%2520Starter%2520Files.zip&sa=D&source=editors&ust=1674689684407550&usg=AOvVaw0VZNngPeWbs-agllbPkoWu)

Once the files have been downloaded, extract them from their folder.

3. Move the assets into your project
